# Portfólio profissional

Este é um portfólio estático criado para divulgar trabalhos, projetos e evolução profissional.

## Como editar

- Troque `Seu Nome`, `SN`, e os links de contato em `index.html`.
- Atualize os cards em `Projetos em destaque` com seus repositórios reais.
- Ajuste métricas como `+10 projetos`, `3x mais agilidade` e `100% foco` para números reais.
- Substitua `assets/workspace-visual.svg` por uma foto, captura de tela ou arte própria se quiser.

## Como publicar no GitHub Pages

1. Crie ou abra o repositório no GitHub.
2. Envie os arquivos desta pasta para a pasta `portifolio` ou para a raiz do repositório.
3. No GitHub, acesse `Settings > Pages`.
4. Escolha a branch principal e a pasta onde está o `index.html`.
5. Salve e aguarde o link de publicação aparecer.

## Estrutura

```text
portfolio/
  index.html
  styles.css
  script.js
  assets/
    workspace-visual.svg
```
